# 1. Two Sum

🔗 https://leetcode.com/problems/two-sum/

**Сложность:** Easy

## Условие

Дан массив целых чисел `nums` и целое число `target`. Верните индексы двух чисел,
сумма которых равна `target`.

Гарантируется ровно одно решение; один и тот же элемент дважды использовать нельзя.
Порядок индексов в ответе не важен.

## Пример

```
Вход:  nums = [2,7,11,15], target = 9
Выход: [0,1]   (nums[0] + nums[1] == 9)
```

## Идея

Один проход с хеш-таблицей `значение → индекс`: для каждого `n` проверяем, видели ли
ранее `target - n`. Время и память — O(n).
